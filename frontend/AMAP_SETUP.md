# 高德地图 API 集成说明

## 获取高德地图 API Key

1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 注册并登录账号
3. 进入「控制台」→「应用管理」→「我的应用」
4. 点击「创建新应用」
5. 创建「Web端(JS API)」类型的 Key
6. 填写应用名称,选择 Web 端(JS API)
7. 配置域名白名单(开发阶段可留空)
8. 复制生成的 Key

## 配置 API Key

在 `.env.development` 文件中配置:

```env
VITE_AMAP_KEY=your_amap_key_here
```

将 `your_amap_key_here` 替换为你获取的高德地图 API Key。

## Mock 数据

当前项目已启用 Mock 数据模式(`VITE_USE_MOCK=true`),可以在没有后端的情况下运行和测试前端功能。

Mock 数据包含:
- 模拟用户登录/注册
- 模拟故事列表
- 模拟地图标记
- 模拟随机漫步
- 模拟故事发布

## 开发调试

### 启动开发服务器

```bash
npm run dev
```

### 访问应用

打开浏览器访问 `http://localhost:5173`

### 测试流程

1. 访问首页,点击「开始探索」
2. 在登录模态框中输入任意邮箱和密码(如: test@test.com / 123456)
3. 登录成功后跳转到地图页
4. 浏览地图上的故事标记
5. 点击标记查看故事详情
6. 点击右侧切换按钮显示/隐藏故事列表
7. 点击「+」按钮发布新故事
8. 点击「🎲」按钮随机漫步

## 注意事项

1. 高德地图 API Key 需要配置才能正常显示地图
2. 如未配置 API Key,地图将显示为空白
3. Mock 数据模式下,所有操作都在内存中进行,刷新页面会重置
4. 图片上传功能当前使用 Blob URL,刷新后失效
5. 地理位置 API 需要用户授权才能使用
