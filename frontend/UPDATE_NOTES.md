## 📝 使用说明

### 地图功能

#### 切换地图主题
1. 进入地图页面
2. 点击右上角的 ☀️/🌙 按钮
3. 地图会立即切换到对应主题
4. 主题选择会自动保存

#### 查看故事标记
- 地图上的星星标记表示用户发布的故事
- 星星颜色代表不同的情绪
- 悬停星星会放大显示
- 点击星星查看故事详情

#### 星星标记说明
- ⭐ 星星形状: 用户发布的位置
- 😊/😢/😐/🤩/😌: 中心的情绪图标
- 🔒 锁定状态: 时光胶囊未解锁
- 持续闪烁: 增加视觉吸引力

### 图片上传功能

#### 方式一: 点击上传
1. 点击"添加图片"按钮
2. 选择一张或多张图片
3. 图片会自动显示在预览区域

#### 方式二: 拖拽上传
1. 将图片文件拖到上传区域
2. 松开鼠标完成上传
3. 支持一次性拖拽多张图片

#### 图片操作
- **预览**: 点击图片查看大图,或点击 👁️ 按钮
- **旋转**: 点击 🔄 按钮旋转图片(每次90度)
- **删除**: 点击右上角 × 按钮删除图片
- **浏览**: 在预览模式下,使用左右箭头切换图片

## 💡 技术实现

```javascript
// 切换地图样式
map.setMapStyle(
  isDarkMode ? 'amap://styles/dark' : 'amap://styles/whitesmoke'
);

// 重新创建标记以更新颜色
updateMarkers();
```

### 星星标记实现
```javascript
// 自定义标记内容
content.innerHTML = `
  <div class="marker-wrapper">
    <div class="marker-star">⭐</div>
    <div class="marker-emotion">${emotion}</div>
  </div>
`;
```

### 拖拽上传实现
```javascript
// 拖放事件处理
@dragover.prevent="isDragOver = true"
@dragleave.prevent="isDragOver = false"
@drop.prevent="handleDrop"
```

## 🔧 配置说明

### 环境变量
无需修改配置,所有功能开箱即用。

### 地图主题默认值
可以在 `localStorage` 中修改默认主题:
```javascript
localStorage.setItem('mapTheme', 'dark'); // 深色模式
localStorage.setItem('mapTheme', 'light'); // 浅色模式
```

## ⚠️ 注意事项

### 图片限制
- 最多上传 9 张图片
- 每张图片不超过 10MB
- 支持格式: JPG, PNG, WebP

### 性能建议
- 大图片会自动压缩
- 预览使用 Blob URL,刷新后失效
- 建议使用合理大小的图片(2-5MB)

### 主题切换
- 主题会保存到本地存储
- 清除浏览器缓存会重置主题
- 切换主题会重新加载标记
